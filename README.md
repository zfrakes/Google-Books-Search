# Google-Books-Search

## Overview

Google Books Search is an application where a user can search for books. Then have the option to view , save and or delete a books.  


## Technologies and Languages Used 
* mongo 
* mongoose 
* Robo 3T
* JavaScript 
* Bootstrap/ React Bootstrap
* React
